Datos del pedido

Cantidad a emplear

Especifica la cantidad que deseas utilizar para la operación actual.

Tiempo de Impresión

Introduce la duración estimada del proceso de impresión. Asegúrate de especificar el tiempo en horas (h).

Ganancia

Introduce el porcentaje de ganancia estimado o deseado para la operación.

Tasa de Fallo

Introduce el porcentaje de fallos. Asegúrate de que el valor ingresado esté expresado en porcentaje.

Coste Operario

Introduce el costo asociado al operario o mano de obra involucrada en el proceso.
